Installing npm (if powershell/cmd displays "npm : The term 'npm' is not recognized as the name of a cmdlet, function, script file, or operable program.")

Download and install: https://nodejs.org/en/

npm install bootstrap --save
Node-sass is not recognized by command line - You need to do npm install -g node-sass to install it globally
execution polition - Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Unrestricted

Importing bootstrap library

Terminal(powershell) - type in "sass filename.scss filename.css". Best to install Live Sass Compiler from extension

at terminal(powershell), type in "npm install bootstrap"

at .scss file, type in "@import "node_modules/bootstrap/scss/bootstrap";"

Issues/notes

Bootstrap 5.2 dropdown nav menu not working

Was playing with dropdown nav menu and nothing happens when i clicked on the drop. After some searches i noticed is something related to the bootstrap scipts. Important to make sure the imported bootstrap scripts are of the same version as the installed bootstrap (?)
Make sure that any css changes to be made above the "@import "node_modules/bootstrap/scss/bootstrap";". otherwise it wouldnt work

Spent quite a while trying to make an animated drop down menu, the kind that slides down when clicked and slides back up when clicked again. Turns out that it did not work because i did not include the js script below: 

                    <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
Refer to html and css file for the code block above
To import bootstrap from root folder into css file when making new folders (like webpages), use the path below: 

                    @import "../node_modules/bootstrap/scss/bootstrap";
